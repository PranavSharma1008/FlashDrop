{% if page_id == "features" %}

FlashDrop is built for fast, simple, local file sharing between devices on the same network.

  • Direct P2P transfer — Files move device-to-device over TCP sockets with no cloud server.
  • Large file support — Chunked streaming handles big files safely with progress tracking.
  • Multi-file queue — Upload, select, and send multiple files from an outbox in one flow.
  • Drag and drop upload — Drop files straight into the app to add them to your outbox.
  • Live transfer stats — Real-time speed, percentage, and ETA while sending or receiving.
  • Receiver inbox — All received files stay in one place until you save or delete them.
  • QR connect — Scan or share a QR code to connect peers quickly without typing IPs.
  • Peer security controls — Optional PIN, allow/deny unknown peers, and cleanup options.
  • Modern UI — Clean cards, dark mode, toast notifications, and responsive layout.
  • Works on your LAN — Share files across Mac, Windows, and other devices on the same Wi-Fi.

{% elif page_id == "about" %}

FlashDrop is a local peer-to-peer file sharing app inspired by tools like SHAREit, built for people who want fast transfers without uploading to the cloud.

It runs a small Flask web server on your machine and uses direct TCP socket transfers between devices on the same network. One device uploads files to its outbox, connects to another device by IP, and sends selected files straight across.

On the receiving side, files appear in the Receiver Inbox where you can save them to your Downloads folder, delete them, or manage multiple files at once.

FlashDrop focuses on speed, simplicity, and privacy: your files stay on your local network, transfers show live progress, and the interface is designed to be easy even for beginners.
